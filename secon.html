• 포트폴리오 팀원소개 SECON2025

지난 3월 19일부터 3월 21일까지 진행한 세계 보안 엑스포(SECON) 2025를 다녀온 후기입니다.

사전예약을 통해 무료로 입장할 수 있었습니다.

SK shieldus의 “No-Code 기반 SOAR” 세션에서 소개한 자동화된 보안 운영 방식이 신기했고,
No-Code로 보안 프로세스를 간소화하는 게 흥미로웠습니다.

AhnLab의 “AhnLab Cloud Plus” 세션에서는 클라우드 보안의 실제 활용 사례와,
기업 환경에서 어떻게 적용되는지 구체적으로 알 수 있었습니다.

SECON 2025는 보안 기술의 최신 트렌드를 보고, 업계 사람들과 교류할 수 있기 때문에
보안 분야에 관심 있는 대학생이라면 꼭 한번 가볼 만한 행사라고 생각합니다!
